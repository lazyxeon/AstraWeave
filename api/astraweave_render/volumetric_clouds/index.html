Volumetric Clouds — Perlin-Worley Raymarching Pipeline

Implements physically-based volumetric clouds following Schneider & Vos (SIGGRAPH 2015) “The Real-Time Volumetric Cloudscapes of Horizon Zero Dawn.”

Architecture (2-pass compute pipeline):

  1. Raymarch (half-resolution): March through cloud shell, sample Perlin-Worley noise for density, compute Beer-Powder lighting with dual-lobe HG phase. Temporal blend with history for stability.
  2. Composite (full-resolution): Bilinear upscale from half-res, depth-aware compositing over the scene (clouds are behind geometry).

Cloud noise is fully procedural (no precomputed noise textures):

  • Base shape: 3-octave FBM of Perlin-Worley noise
  • Detail erosion: 2-octave Worley FBM
  • Coverage: analytical weather function modulated by cloud_coverage
