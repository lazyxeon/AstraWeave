The step count adapts\n// based on view angle: grazing angles use more steps for accuracy.\n//\n// Usage: Import these functions from your PBR shader. SINGLE TEXTURE (standard PBR materials)\n// ============================================================================\n\n/// Compute parallax-displaced UV coordinates using steep parallax + binary refinement.\n///\n/// Parameters:\n///   height_tex  - single-channel (R) heightmap texture (1.0 = raised, 0.0 = depressed)\n///   height_samp - sampler for the heightmap\n///   uv          - initial texture coordinates\n///   view_ts     - view direction in tangent space (T, B, N basis)\n///   height_scale - maximum displacement depth in UV space (typical: 0.02\u{2013}0.08)\n///\n/// Returns: mid_height;\n        } else {\n            // Past the surface \u{2014} search shallower\n            current_uv = mid_uv;\n            current_layer_depth = mid_depth;\n            current_height = mid_height;\n        }\n    }\n\n    return current_uv;\n}\n\n/// Compute self-shadowing from a heightmap (optional \u{2014} called after POM offset).\n///\n/// Returns a shadow factor (0.0 = fully
