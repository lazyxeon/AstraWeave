Temporal Upsampling (TAA-U) — Custom Temporal Upscaler

Renders at a configurable internal resolution (50-100% of display), then accumulates sub-pixel detail at native display resolution over time using TAA-style temporal reprojection. Each frame’s jittered sub-pixel offset samples a unique position within the display texel, gradually building native-resolution detail in the history buffer.

Pipeline: reduced-res render → upscale resolve → RCAS sharpen → output

Integrates with the existing TAA jitter infrastructure (super::taa).
