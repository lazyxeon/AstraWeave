Module brdf_lut

Source
Expand description

BRDF Integration LUT for split-sum IBL approximation.

Precomputes a 2D lookup table parameterized by (NdotV, roughness) that stores the integral of the GGX BRDF split into (scale, bias) terms: specular = prefiltered_color * (F0 * scale + bias)

This LUT is resolution-independent and only needs computation once (or when sample count changes).
