Module snow_footprint

Expand description

Snow footprint stamp system — entity deformation of accumulated snow.

Stamps entity foot positions into a snow depression map that is subtracted from the snow accumulation during PBR evaluation. Footprints slowly recover (snow settling) via a per-frame recovery pass.

Uses the same world-space top-down texture approach as vegetation_interaction.
