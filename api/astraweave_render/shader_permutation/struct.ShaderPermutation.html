Struct ShaderPermutation

Source 
pub struct ShaderPermutation(/* private fields */);
Expand description

Shader permutation flags — each bit enables a Disney BRDF lobe.

When a flag is NOT set, the corresponding WGSL const is false and the GPU compiler eliminates the branch entirely (no warp divergence).

pub fn generate_defines(&self) -> String

Generate the WGSL const-defines preamble for this permutation.

Produces lines like:

const ENABLE_CLEARCOAT: bool = true;
const ENABLE_ANISOTROPY: bool = false;

The GPU compiler statically eliminates branches gated on false consts, avoiding warp divergence entirely.

Source

pub fn generate_disney_brdf(&self) -> String

Generate a complete Disney BRDF shader source with permutation defines.

The result includes constants.wgsl + permutation defines + disney_brdf.wgsl. Unused lobes are compiled out by the GPU compiler.
