Module vegetation_gpu

Source
Expand description

GPU-instanced vegetation scatter and frustum cull pipeline.

Two compute passes:

  1. Scatter: generates vegetation instances from a heightmap + biome data using a deterministic jittered-grid placement (GPU Poisson disk analog).
  2. Cull: frustum-culls the generated instances and writes survivors into a compacted draw-indirect buffer for instanced rendering.

The CPU scatter path in astraweave-terrain remains available as a fallback.
