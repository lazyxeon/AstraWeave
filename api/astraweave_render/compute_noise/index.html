Module compute_noise

Source
Expand description

GPU compute noise generation for terrain heightmaps.

Generates Perlin/fBM/Ridged/Billow/DomainWarped noise entirely on the GPU, outputting directly to an R32Float texture. Eliminates the CPU→GPU upload bottleneck for terrain noise evaluation.

§Usage

// Configure noise for a 512×512 terrain chunk
let config = GpuNoiseConfig {
    resolution: [512, 512],
    frequency: 0.01,
    amplitude: 1.0,
    octaves: 6,
    noise_type: GpuNoiseType::Fbm,
    ..Default::default()
};

§Performance

512×512 chunk at 6 octaves: ~0.5ms on GTX 1660 Ti (vs ~8ms CPU + upload).
