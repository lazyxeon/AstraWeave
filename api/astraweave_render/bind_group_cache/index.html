Module bind_group_cache

Source
Expand description

Bind group cache to eliminate per-frame device.create_bind_group() overhead.

Many render passes create identical bind groups every frame because their input textures and buffers haven’t changed. This module provides a generation-based cache: bind groups are rebuilt only when the renderer signals a resource change (e.g., resize, texture swap) by bumping the generation counter.
