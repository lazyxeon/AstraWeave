BiomePack: a data-driven format for importing decomposed .blend scene assets as biome vegetation/scatter profiles.

A BiomePack is created from the manifest.json produced by the scene decomposition pipeline (astraweave-blend) and describes all assets, their categories, textures, and scatter parameters needed to generate a custom biome.
