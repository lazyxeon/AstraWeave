pub fn compute_pixel_error(
    geometric_error: f32,
    distance: f32,
    view: &ViewParams,
) -> f32
Expand description

Compute the projected pixel error given a world-space geometric error, distance to the observer, and view parameters.

Formula:

pixel_error = (geometric_error * screen_height) / (distance * 2.0 * tan(fov_y / 2.0))

Returns f32::MAX when distance is effectively zero.
