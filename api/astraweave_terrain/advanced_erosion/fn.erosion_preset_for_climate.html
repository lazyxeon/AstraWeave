pub fn erosion_preset_for_climate(climate: ClimateBias) -> ErosionPreset
Expand description

Phase 1.6-F.3-phase-2.A: selects an ErosionPreset for a chunk based on its climate bias. Per campaign plan §2.2, the six ClimateBias variants map to four distinct presets (two 2-to-1 groupings: Cold / Highland → mountain-family, Tropical / Wetland → coastal).

Temperate and Cold / Highland use “balanced” preset variants with reduced droplet counts to stay inside §2.3’s 30-second generation budget for 121-chunk editor runs (F.3-phase-0 projected full default at 39.7s and full mountain at 83.5s).
