First time here? Try Tour → First-time onboarding for a 10-minute guided walk-through (eight steps, narration + reconfigured graph at each step).

Click any node for crate detail. The side panel uses progressive disclosure: surface info visible, deeper sections (deps, integration types, open questions, silent failures) behind ▸ expanders. Blast radius (toggle, default on): selecting a node highlights its dependents within 3 hops — orange = direct, faded = transitive. Honest dependency-graph highlighting, not behavioral simulation. Focus mode (toggle, off by default): hides everything outside the selected node's 1-hop neighborhood. Use when the hairball is too dense. Stories dropdown for narrative views (Dual-World Bridge, Editor↔Engine Boundary, Silent-Failure Map, Dormant Surface, AI Pipeline, Networking Coexistence, Aspirational Docs). Path-find: Pick A → node → Pick B → node → Show path highlights the shortest directed dep chain. Share ↗ copies a URL that reproduces the exact current view (filters, selection, tour step, layout). Domain chips (v2.1): click any chip to add that domain to the include set. Nothing checked = no filter (show all). Any subset checked = show only those domains. All checked = same as none. Chips lock during a story or tour — use Reset to clear. Visual encoding: node size = LoC band; fill colour = domain (Wong colourblind-safe palette); dashed border = dormant; ⚠ = silent-failure locus; 📄 = traced. Edges (v2.1) are linear gradients from source-domain colour to target-domain colour — same-domain edges look uniform, cross-domain edges show the transition. Anomalous edges remain solid red; declared-but-unused edges remain dashed grey. Keyboard: Esc to clear; ← → to navigate tour steps. On mobile, tap ☰ Controls for the toolbar; the side panel becomes a bottom sheet.
